Это мой первый проект
v.1.1

Программа представляет из себя консольный калькулятор для двух операндов.
Реализованы следующие математические функции: сложение, вычитание, умножение, деление и возведение в степень
Содержит в себе классы: Calculator, Main, Operand.
Содержит Unit Tests: CalculatorTests, OperandTests.
Ссылка проекта на GitHub: https://github.com/Doc813/LabISRPO
